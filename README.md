# Voice Assistant Việt - Đồ án Chuyên Ngành

## Hướng dẫn chạy Demo

1. **Backend (Python):**
   - Vào thư mục `Backend`
   - Chạy: `pip install -r requirements.txt`
   - Chạy: `python api/server.py`

2. **Mobile App:**
   - Vào thư mục `UIVoiceAssistant`
   - Chạy: `npm install`
   - Chạy: `npx cap run android` (kết nối điện thoại thật)
